Figma

Trello
Trello is used to keep track of all tasks that need to be done for this assignment. It is organised into 4 sections:
  • To start – Tasks that has not yet been started
  • To Complete – Tasks that are assigned to member(s) and being worked on
  • Done – Tasks that have been finished
  • Shelved – Tasks that have been dropped and no longer planned to complete
