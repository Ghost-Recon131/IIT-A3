Skills and Jobs


Software Developer
Software development is a vital portion of this project when it comes to building and implementing the functional programs for the parts to work together. This project is currently looking for an experienced software developer to join the programming team to create and maintain the software used by other members on the project. Candidates for this role will primarily focus on developing the appropriate software for different scenarios within the project and to help support other members with the functionalities of the software. Working as a software developer will require a constant degree of upkeep to ensure that software performs efficiently and to resolve errors.

The skills and responsibilities for this role:
  • Confident experience using several languages such as Java, Python, C++, etc.
  • Basic knowledge of Git, SQL , APIs
  • Basic knowledge of unit testing
  • An understanding of data structure and algorithms for creating optimal and efficient programs
  • Responsible for the creation of multiple types of software
    • AI detection modules for endpoints/servers potentially built in C++ for speed and programmer managed memory
    • Software development kits for developers built either Java or Python for wide use
    • Software for controlling the machine learning process built on Python for extended library of machine learning tools
  • Having experience in user experience and font development would be considered an asset to creating an appropriate front end for end users utilising the software
  • Must have the ability to communicate clearly and effectively to assist in collaborations with partners within the project
Data Scientist
The project requires an adequate understanding of utilisation and manipulation of large quantities of data to assist in the development of AI models and machine learning. This project is currently looking to employ an experienced data scientist within the data science team. The role will be responsible for providing specialist expertise and data insights while also developing the appropriate software and tools while working with other teams in the project. The ideal candidate for this role is someone with a fair amount of experience working with data related problems and has strong technical skills.

The skills and responsibilities for this role:
  • Confident experience using languages such as Python and SQL (knowing Java and C++ is also useful)
  • Data Literacy is a mandatory requirement to ensure that the candidate is capable of manipulating, analysing, and visualising the data given to them
  • Knowledgeable on machine learning and AI. A significant portion of the data will focus on using machine learning and AI modelling to help train programs in circumventing attacks and candidates must be capable of utilising such tools.
  • Knows how to adjust learning algorithm and hyperparameter optimisation
  • Must have the ability to communicate clearly and effectively to assist in collaborations with partners within the project
  • Will be involved in creating data cleaning software and tools while working alongside the software developer team
Cybersecurity Analyst
This project revolves around building a solid cybersecurity defence and is looking for a cybersecurity analyst to assist the team. The role requires a solid ability of critical thinking and problem solving when it comes to dealing with various issues or attacks that pop up or may potentially occur. Working on this project will help improve these skills and obtain more experience for other skillsets such as risk management and analysis. The role also expects a degree of assistance to other members within the team to ensure that they have adequate knowledge behind the security of the programs and models being developed.

The skills and responsibilities for this role:
  • Risk management is an important skillset for this role to identify, evaluate and resolve any potential risks that have either already occurred or have an established plan for errors that could occur
  • Require an apt degree of critical thinking and problem solving to troubleshoot any issues
  • Capable of performing black box tests on software to find areas of weakness and assessing ways to fix them
  • Must have the ability to communicate clearly and effectively to assist in collaborations with partners within the project
  • Will work alongside the admin team to determine whether certain issues are either real threats or false positives
  • Assist the software development team in creating SDK for other developers within the project
